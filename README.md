## 📋 Exercises Included

### Basic Syntax & Operations
| # | Exercise | Description |
|---|----------|-------------|
| 4 | `greet()` | Function with default parameter |

| 10 | List summation | Sum of numbers in a list |

| 12 | Ternary operator | `"Adult" if age>=18 else "Minor"` |

| 18 | While loop with `continue` | Skips printing number 3 |

### Algorithm Challenges
1. **Variable Swap** - Swap two numbers without a temporary variable
2. **FizzBuzz** - Classic programming challenge (1 to 30)
3. **Grade Calculator** - Function returning letter grades based on marks
4. **Even Numbers Sum** - Sum of all evens from 1 to 100 using `range()`
5. **Factorial Function** - Calculate factorial using while loop
